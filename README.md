
Lancer l'application :  
Avoir python (>=3.8) d'installé  


Serveur :  

Installation :  
`$ cd server`  
`$ source env/bin/activate`  
`(env)$ pip install -r requirements.txt`  

Lancement du server : avec env activé, lancer :  
`(env)$ flask run --port=5001 --debug`  


Client :  

Installation :  
Dans un nouveau terminal :  
`$ cd client`  
`$ npm install`  

Lancement du client :  
`$ npm run dev`
